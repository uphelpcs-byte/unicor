4. 공통 자가조치 안내 (응대 스크립트)

전화·채팅 응대 시 우선 안내. 해결 시 상담처리(무상) 종결, 미해결 시 출동 접수 전환.

▸ 9V 비상개문 (외부 잠김 시)

  1. 외부 비상단자에 9V(네모난) 건전지를 접촉한 상태 유지.
  2. 접촉한 채로 비밀번호 입력 후 별표(*).
  3. 점등·개문되면 내부에서 정품 건전지 전체 교체.
  4. 반복 시 키패드/기기 점검 접수.

▸ 리셋 (먹통·터치불량)

  1. 하단 일자형 돌출부 사이 리셋홀 확인.
  2. 핀/옷핀으로 약 5초 눌렀다 떼기.
  3. 건전지 분리 후 재장착, 동작 확인.
  4. 미해결 시 방문 점검.

▸ 데드볼트 걸림 임시개방

  1. 비밀번호 입력 후 별표(*).
  2. 손잡이 잡고 문 앞뒤로 강하게 흔들며 개방.
  3. 반복 시 모티스/걸쇠 점검 접수.

▸ 건전지 교체

  1. 실내측 안전잠금버튼 위 홈을 누르며 위로 밀어 커버 개방.
  2. 알카라인 정품 전체 교체(망간·충전식 금지).
  3. 교체에도 한 달 미만 방전이면 점검 접수.

▸ 자동잠금 재설정

  1. 건전지 정품 전체 교체.
  2. 모델별 자동잠금 설정 재실행(등록버튼+외부버튼).
  3. 문틀 자석/스트라이커 위치 확인.
  4. 안전잠금버튼 무반응이면 고장 → 방문 접수.

▸ 마스터(비밀번호) 등록

  1. 반드시 문을 연 상태에서 진행.
  2. 커버 열고 등록버튼 → 새 번호 → 등록버튼.
  3. 공장 초기 비번 1234, 입력 후 별표(*) 적용.

▸ 지문 등록

  1. 등록버튼 누른 뒤 같은 손가락 4회 연속 인식.
  2. 완료 후 등록버튼으로 종료.
  3. 등록 중 꺼지면 건전지·인식부 확인.

▸ 터치형 자동잠금

  1. 외부 번호판에 손바닥 3초 이상 → 잠김.
  2. 번호 입력 없이 터치만으로 동작.
