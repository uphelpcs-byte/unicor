01 안전을 위한 주의사항

⚠ 안전을 위한 주의사항은 사용자의 안전을 지키고 사고로 인한 재산상의 손해 및 불편을 막기 위한 내용입니다. 반드시 읽고 올바르게 사용해 주십시오.

  • 사용자가 임의로 제품을 설치, 분해, 수리하지 마십시오.
  • 문과 문틀과의 간격이 5mm 이상인 곳에는 설치하지 마십시오.
  • 비밀번호를 타인이 알지 못하도록 주의하여 관리하고, 주기적으로 변경하여 주십시오.
  • 젖은 손으로 제품을 조작하거나 제품 내부에 물, 음료수 등과 같은 액체가 들어가지 않도록 하십시오.
  • 도어록의 버튼을 무리하게 누르거나 날카로운 도구를 사용해 누르지 마십시오.
  • 제품을 청소할 때는 물, 벤젠, 알코올 등으로 닦지 마시고 부드러운 마른 천을 사용하여 주십시오.
  • 건전지는 제품의 극성(+/−)에 맞게 주의하여 정확하게 삽입하여 주십시오. 역삽입 시 누액이나 파열이 발생할 수 있으므로 주의하십시오.
  • 건전지 수명이 다하면 모든 건전지를 교환하시고, 새 건전지와 사용 중이거나 완료된 건전지를 혼용하지 마십시오.

  • 비나 눈 등에 노출되는 장소와 직사광선에 직접 장시간 노출되는 곳, 습기와 염분이 많은 곳에는 설치하지 마십시오.

30초 LOCK 기능

외부에서 도어록을 잘못 조작할 경우에 경고음과 함께 도어록이 작동하지 않도록 차단하는 기능입니다. 잘못된 인증이 연속 5회 반복 입력 시 30초간 경보음이 울리고 30초 동안 도어록 작동이 정지됩니다.

해제 방법 30초간 경보음 발생 시 오류난 인증수단이 아닌 등록된 다른 인증수단으로 인증하거나, 실내측에서 문을 열면 해제됩니다.

잠금장치 걸림 알림 기능

※ 계속해서 경고음이 발생하거나 정상적으로 동작하지 않을 경우 도어록의 위치를 재조정하여야 하므로 설치점 또는 도어록 전문가에 도움을 요청하시기 바랍니다. 이러한 경우 유상서비스 대상이오니 참고하시기 바랍니다.

건전지 교환시기 알림 기능

건전지 교체 시기가 되면 본체 개폐 동작 시 "동, 동, 동대문을 열어라~ 남, 남, 남대문을 열어라~" 하는 멜로디가 울려 건전지 교환시기를 알려줍니다.

교환방법 최초로 경고음이 울린 후, 1주일 이내에 건전지 4개를 모두 새것으로 교환하여 주십시오. (교환하지 않고 계속 사용 시 작동이 멈춥니다.)

• 반드시 충전식이 아닌 일반 건전지를 사용하셔야 합니다. • 건전지 교체 시 일부가 아닌 모든 건전지를 새것으로 교환하셔야 하며, 새 건전지와 방전된 건전지를 혼용하면 안 됩니다. • 정격전압 6V에 동작하게 설계되어 있으며, (4.2V±0.3V) 이하 저전압 감지 시 멜로디가 발생됩니다. • 새 건전지로 교환 후 비정상적으로 짧은 시간 안에 멜로디가 울리거나 방전될 경우 건전지 자체 불량일 수 있으니 건전지 구입처에 문의하여 교환받으시기 바랍니다.

비상전원 공급 기능

제품에 삽입한 건전지가 완전히 방전되어 실외측에서 문을 열 수 없는 경우에 비상전원 공급단자에 9V 건전지를 이용하여 비상전원을 공급할 수 있도록 해주는 기능입니다.

  1. 가까운 문구점이나 편의점에서 직사각형 모양의 9V 건전지를 구입합니다.
  2. 실외측 본체의 비상전원단자에 9V 건전지를 접촉시킨 상태에서 정상적으로 비상전원이 공급되면 "띠리~"하고 번호판에 불이 들어옵니다.
  3. 등록된 비밀번호를 입력하거나 스마트키를 접촉하면 문이 열립니다.

• 9V 건전지는 +,− 극성에 상관없이 접촉하여도 무방합니다. • 9V 건전지는 비상전원단자에 접촉시킨 상태를 유지하고 있어야 비상전원이 공급됩니다.(충전형 아님) • 비상전원 공급 후 즉시 방전된 건전지를 모두 교환해 주시기 바랍니다.

30초 LOCK 기능

외부에서 도어록을 잘못 조작할 경우에 경고음과 함께 도어록이 작동하지 않도록 차단하는 기능입니다. 잘못된 인증이 연속 5회 반복 입력 시 30초간 경보음이 울리고 30초 동안 도어록 작동이 정지됩니다.

해제 방법 30초간 경보음 발생 시 오류난 인증수단이 아닌 등록된 다른 인증수단으로 인증하거나, 실내측에서 문을 열면 해제됩니다.

잠금장치 걸림 알림 기능

문이 열리거나 잠길 때 데드볼트(잠금장치)가 정상적으로 나오거나 들어가지 못할 경우 3회 반복하여 시도하며, 정상 동작 실패 시 15초간 멜로디가 발생합니다.

※ 계속해서 경고음이 발생하거나 정상적으로 동작하지 않을 경우 도어록의 위치를 재조정하여야 하므로 설치점 또는 도어록 전문가에 도움을 요청하시기 바랍니다. 이러한 경우 유상서비스 대상이오니 참고하시기 바랍니다.

건전지 교환시기 알림 기능

건전지 교체 시기가 되면 본체 개폐 동작 시 "동, 동, 동대문을 열어라~ 남, 남, 남대문을 열어라~" 하는 멜로디가 울려 건전지 교환시기를 알려줍니다.

교환방법 최초로 경고음이 울린 후, 1주일 이내에 건전지 4개를 모두 새것으로 교환하여 주십시오. (교환하지 않고 계속 사용 시 작동이 멈춥니다.)

• 반드시 충전식이 아닌 일반 건전지를 사용하셔야 합니다. • 건전지 교체 시 일부가 아닌 모든 건전지를 새것으로 교환하셔야 하며, 새 건전지와 방전된 건전지를 혼용하면 안 됩니다. • 정격전압 6V에 동작하게 설계되어 있으며, (4.2V±0.3V) 이하 저전압 감지 시 멜로디가 발생됩니다. • 새 건전지로 교환 후 비정상적으로 짧은 시간 안에 멜로디가 울리거나 방전될 경우 건전지 자체 불량일 수 있으니 건전지 구입처에 문의하여 교환받으시기 바랍니다.

비상전원 공급 기능

제품에 삽입한 건전지가 완전히 방전되어 실외측에서 문을 열 수 없는 경우에 비상전원 공급단자에 9V 건전지를 이용하여 비상전원을 공급할 수 있도록 해주는 기능입니다.

  1. 가까운 문구점이나 편의점에서 직사각형 모양의 9V 건전지를 구입합니다.
  2. 실외측 본체의 비상전원단자에 9V 건전지를 접촉시킨 상태에서 정상적으로 비상전원이 공급되면 "띠리~"하고 번호판에 불이 들어옵니다.
  3. 등록된 비밀번호를 입력하거나 스마트키를 접촉하면 문이 열립니다.

• 9V 건전지는 +,− 극성에 상관없이 접촉하여도 무방합니다. • 9V 건전지는 비상전원단자에 접촉시킨 상태를 유지하고 있어야 비상전원이 공급됩니다.(충전형 아님) • 비상전원 공급 후 즉시 방전된 건전지를 모두 교환해 주시기 바랍니다.

※ 계속해서 경고음이 발생하거나 정상적으로 동작하지 않을 경우 도어록의 위치를 재조정하여야 하므로 설치점 또는 도어록 전문가에 도움을 요청하시기 바랍니다. 이러한 경우 유상서비스 대상이오니 참고하시기 바랍니다.

건전지 교환시기 알림 기능

건전지 교체 시기가 되면 본체 개폐 동작 시 "동, 동, 동대문을 열어라~ 남, 남, 남대문을 열어라~" 하는 멜로디가 울려 건전지 교환시기를 알려줍니다.

교환방법 최초로 경고음이 울린 후, 1주일 이내에 건전지 4개를 모두 새것으로 교환하여 주십시오. (교환하지 않고 계속 사용 시 작동이 멈춥니다.)

• 반드시 충전식이 아닌 일반 건전지를 사용하셔야 합니다. • 건전지 교체 시 일부가 아닌 모든 건전지를 새것으로 교환하셔야 하며, 새 건전지와 방전된 건전지를 혼용하면 안 됩니다. • 정격전압 6V에 동작하게 설계되어 있으며, (4.2V±0.3V) 이하 저전압 감지 시 멜로디가 발생됩니다. • 새 건전지로 교환 후 비정상적으로 짧은 시간 안에 멜로디가 울리거나 방전될 경우 건전지 자체 불량일 수 있으니 건전지 구입처에 문의하여 교환받으시기 바랍니다.

  1. 가까운 문구점이나 편의점에서 직사각형 모양의 9V 건전지를 구입합니다.

05 비상경고 기능 — 화재대비 고온감지

실내의 온도가 비정상적으로 올라가는 경우 온도센서가 감지하여 경보를 울리는 기능입니다. 고온경보 기능은 70℃ ± 10℃ 이상일 경우 경보 발생 5초 후 자동으로 문이 열리고 80dB 상당의 경보음이 3분간 발생하며, 경보가 발생하면 내부 안전잠금 및 모든 잠금 설정과 관계없이 자동으로 잠금해제가 됩니다.

※ 제품온도가 높은 상태에서는 건전지 제거 시에 경고음 해제됩니다. ※ 실내 온도가 일정 온도 이하로 낮아지면 도어록이 정상으로 작동합니다.

침입경보 / 파손경보 기능

외부나 내부에서 강제적으로 문을 열 경우 경보음을 발생시키는 기능입니다. 7초 이상 문이 닫힌 상태에서 강제로(예: 문틈을 벌려서) 문이 열려 문 열림 센서가 이를 감지하면 80dB 상당의 경보음이 3분간 발생합니다.

30초 LOCK 기능

외부에서 도어록을 잘못 조작할 경우에 경고음과 함께 도어록이 작동하지 않도록 차단하는 기능입니다. 잘못된 인증이 연속 5회 반복 입력 시 30초간 경보음이 울리고 30초 동안 도어록 작동이 정지됩니다.

해제 방법 30초간 경보음 발생 시 오류난 인증수단이 아닌 등록된 다른 인증수단으로 인증하거나, 실내측에서 문을 열면 해제됩니다.

잠금장치 걸림 알림 기능

문이 열리거나 잠길 때 데드볼트(잠금장치)가 정상적으로 나오거나 들어가지 못할 경우 3회 반복하여 시도하며, 정상 동작 실패 시 15초간 멜로디가 발생합니다.

※ 계속해서 경고음이 발생하거나 정상적으로 동작하지 않을 경우 도어록의 위치를 재조정하여야 하므로 설치점 또는 도어록 전문가에 도움을 요청하시기 바랍니다. 이러한 경우 유상서비스 대상이오니 참고하시기 바랍니다.

건전지 교환시기 알림 기능

건전지 교체 시기가 되면 본체 개폐 동작 시 "동, 동, 동대문을 열어라~ 남, 남, 남대문을 열어라~" 하는 멜로디가 울려 건전지 교환시기를 알려줍니다.

교환방법 최초로 경고음이 울린 후, 1주일 이내에 건전지 4개를 모두 새것으로 교환하여 주십시오. (교환하지 않고 계속 사용 시 작동이 멈춥니다.)

• 반드시 충전식이 아닌 일반 건전지를 사용하셔야 합니다. • 건전지 교체 시 일부가 아닌 모든 건전지를 새것으로 교환하셔야 하며, 새 건전지와 방전된 건전지를 혼용하면 안 됩니다. • 정격전압 6V에 동작하게 설계되어 있으며, (4.2V±0.3V) 이하 저전압 감지 시 멜로디가 발생됩니다. • 새 건전지로 교환 후 비정상적으로 짧은 시간 안에 멜로디가 울리거나 방전될 경우 건전지 자체 불량일 수 있으니 건전지 구입처에 문의하여 교환받으시기 바랍니다.

비상전원 공급 기능

제품에 삽입한 건전지가 완전히 방전되어 실외측에서 문을 열 수 없는 경우에 비상전원 공급단자에 9V 건전지를 이용하여 비상전원을 공급할 수 있도록 해주는 기능입니다.

  1. 가까운 문구점이나 편의점에서 직사각형 모양의 9V 건전지를 구입합니다.
  2. 실외측 본체의 비상전원단자에 9V 건전지를 접촉시킨 상태에서 정상적으로 비상전원이 공급되면 "띠리~"하고 번호판에 불이 들어옵니다.
  3. 등록된 비밀번호를 입력하거나 스마트키를 접촉하면 문이 열립니다.

• 9V 건전지는 +,− 극성에 상관없이 접촉하여도 무방합니다. • 9V 건전지는 비상전원단자에 접촉시킨 상태를 유지하고 있어야 비상전원이 공급됩니다.(충전형 아님) • 비상전원 공급 후 즉시 방전된 건전지를 모두 교환해 주시기 바랍니다.

01. APP 다운로드 / 설치

  1. 애플 App Store 또는 안드로이드 Google Play에서 스마트유니코 또는 SMART UNICOR를 검색합니다.
  2. SMART UNICOR(스마트유니코) 앱을 스마트폰에 설치합니다.
  3. 최초 설치 시 필요한 기능(위치·블루투스·카메라·알림 등) 사용을 허용/확인합니다.

02. 기기 인증

  1. 국가코드(+82)와 인증받을 휴대폰 번호를 입력 후 [확인].
  2. 문자(SMS)로 수신된 인증번호를 입력하여 인증을 진행합니다. ※ 휴대폰 기종에 따라 인증번호가 자동 기입될 수 있습니다.
  3. 인증 완료 후 앱 메인으로 접속됩니다.

03. 회원가입 및 로그인

  1. 우측 하단 [더보기] 버튼을 선택합니다.
  2. "로그인이 필요합니다"를 선택하면 회원가입/로그인 창이 나타납니다.
  3. 하단의 회원가입을 진행하여 로그인합니다.

04. 도어락 등록하기

⚠ 앱 설치 후 휴대폰의 블루투스 모드를 실행한 후 진행합니다. · 도어락 등록 시 반드시 문을 열어놓은 상태에서 진행합니다. · 스마트폰을 도어락에 최대한 가까이(5~10cm) 근접하여 실행합니다.

  1. 앱 하단 [더보기] → [도어락 등록하기]를 선택합니다.
  2. 우측 상단의 [+] 버튼을 선택합니다.
  3. [블루투스 검색]을 선택합니다.
  4. 연결 가능 장치에 표시된 등록할 도어락을 선택합니다.
  5. 도어락 이름(예: 우리집)을 지정한 후 [저장하기]를 선택합니다.

05. 동작 방법 — 방법 1: 아이콘 클릭 모드

앱을 실행하고 도어락에 근접 시 블루투스로 자동 연결됩니다. 연결 성공 시 자물쇠 아이콘 안에 블루투스 마크와 "연결됨"이 표시되고 잠금 상태가 보여집니다. 자물쇠 아이콘을 클릭하면 문이 열립니다. ※ 최초 등록 시 화면을 당겨 새로고침 후 파란색 원으로 연결됨을 확인하세요.

05. 동작 방법 — 방법 2: 퀵 오픈 모드

[더보기] → [일반 설정]에서 "퀵 오픈 모드 활성화"를 켭니다. 도어락 근접 후 앱을 실행하면 도어락이 자동으로 열립니다.

05. 동작 방법 — 방법 3: 위젯 오픈 모드

스마트폰 바탕화면의 빈 공간을 길게 눌러 위젯 추가에서 "SMART UNICOR" 위젯을 추가합니다. 도어락 근접 후 위젯을 클릭하면 자동으로 열립니다. ※ 안드로이드는 상단 바에 고정되어 상단 바에서 클릭으로 열 수 있습니다.

05. 동작 방법 — 방법 4: 스마트 패스 모드

[설정] → 스마트패스에서 "기능 활성화"와 "진입/이탈 알림설정"을 켭니다. 외출 후 도어락 근접 시 자동으로 열립니다. ※ 위치정보를 '항상'으로 허용해야 하며, 환경·기종·OS에 따라 변동성이 있는 베타 기능입니다. (200~300M 이상 벗어난 후 동작)

05. 동작 방법 — 방법 5: 와이파이 겸용 원거리 브릿지 오픈 모드

⚠ 통신사 네트워크 및 와이파이망에 정상 연결, 반드시 2.4GHz Wi-Fi 필요. 브릿지는 USB 형태 별도판매이며 어댑터와 결합해 콘센트에 연결 후 등록합니다.

브릿지 등록 [더보기]→[브릿지(WiFi) 등록하기]→[+] → 블루투스 검색 → 브릿지명 수정 후 [저장하기] → 와이파이 이름·비밀번호 입력 후 [확인].
브릿지·도어락 연동 [더보기]→[도어락 등록하기] 접속 후 연결할 도어락 클릭 → [브릿지 선택]에서 연결할 브릿지 선택 후 [저장하기]. 연결되면 도어락 앞에 와이파이 아이콘 표시.
와이파이로 동작 브릿지 연결 성공 시 "연결됨" 표시 → 자물쇠 아이콘 클릭으로 원거리에서 문 열기.
